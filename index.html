Plugin demo for jQuery sticky table headers

How to use this plugin

Make sure you pull jquery.js and sticky.jquery.js OR sticky.jquery.min.js (less than 1 kb) in the HEAD area of your HTML document. Assign a id to the table you'd want to have sticky table headers enabled and call 'sticky' on it from inside document.ready: 

                    <script type="text/javascript" src="jquery.js"></script>
                    <script type="text/javascript" src="sticky.jquery.min.js"></script>
                    <script type="text/javascript">
                        $(document).ready(function(){
                            $("#myTable").sticky();
                        });
                    </script>
